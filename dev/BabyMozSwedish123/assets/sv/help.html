Hjälp

Dra ett finger över skrärmen åt vänster för att bläddra framåt, till nästa siffra. Dra åt höger för att bläddra bakåt, till föregående siffra.

Under inställningra kan du aktivera ljud, animeringar och slumpvis uppspelning:

  • Ljud lär dig hur siffrorna uttalas på svenska.
  • Animationer gör det roligare genom att siffrorna rör sig på skärmen.
  • Siffrorna kan visas i slumpmässig ordning.
