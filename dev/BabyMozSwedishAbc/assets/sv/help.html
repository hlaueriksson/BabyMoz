Hjälp

Dra ett finger över skrärmen åt vänster för att bläddra framåt, till nästa bokstav. Dra åt höger för att bläddra bakåt, till föregående bokstav.

Under inställningra kan du aktivera ljud, animeringar och slumpvis uppspelning:

  • Ljud lär dig hur bokstäverna uttalas på svenska.
  • Animationer gör det roligare genom att bokstäverna rör sig på skärmen.
  • Bokstäverna kan visas i slumpmässig ordning.
